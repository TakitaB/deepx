About the project

Deep X is a GAN absed Neural Network whose aim is to generate fashionable item of clothing, starting from a simple input image. It is a tool used to create new designs based off of the top leading fashion trends. To start using DeepX, upload an image of the the item of clothing you wish to generate similar items of clothing from, and wath the Neural Network do its magic!
